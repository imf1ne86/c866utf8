# c866utf8

Данная программа, написанная на языке программирования MS-DOS QBasic, ver. 1.0, представляет собой утилиту, которая конвертирует текст с MIME-типом "plain/text" из кодировки CP866 (она же - Windows-1251) в кодировку UTF-8 (она же - 65001).

Программа после запуска запрашивает имя входящего и имя исходящего файлов, после чего, собственно говоря, и выполняется конвертация содержимое входных данных из одной кодировки в другую.

В исходный код программы встроена таблица преобразований (она же - таблица соответствий) для работы с кодировкой UTF-8.

Исходный код программы должен храниться в кодировке CP866, поскольку сама программа предназначена для работы из-под операционной системы DOS.
